# 🛍️ Multi-Dimensional Retail Analytics on Istanbul Shopping Data

## 📌 Project Overview

This project performs a multi-dimensional analysis of shopping data from Istanbul malls.
The goal is to explore customer behavior, spending patterns, and revenue trends across different dimensions such as category, shopping mall, gender, and time.

---

## 🎯 Objectives

* Perform multi-dimensional analysis using pandas
* Analyze trends across time, category, and demographics
* Generate actionable business insights

---

## 📊 Dataset

* 99,457 transactions
* 10 columns
* No missing values
* Covers 10 shopping malls in Istanbul

---

## ⚙️ Tools & Technologies

* Python
* pandas
* matplotlib
* seaborn

---

## 🔍 Analysis Performed

* Data cleaning and preprocessing
* Feature engineering (year, month, revenue)
* Revenue analysis by category
* Pivot table (shopping mall vs category)
* Yearly trend analysis (2021–2023)
* Gender-based spending comparison
* Payment method analysis
* Monthly growth rate (bonus)

---

## 📈 Key Insights

* Clothing is the top-performing category
* Cash is the most used payment method
* Female customers dominate shopping activity
* Technology has the highest average purchase value
* 2022 is the peak revenue year

---

## 📌 Conclusion

This project provides valuable insights into retail behavior and helps identify opportunities for improving sales, marketing strategies, and customer targeting.
